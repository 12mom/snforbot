# 🫠 سنفور بوت | SnforBot

مرحبًا بكم في بوت **سنفور**!  
بوت دردشة، ترفيه، صور، ألغاز، تسلية، ونظام نقاط 🎮

---

## 💡 المميزات:

- 📋 أوامر مساعدة منظمة
- 🎯 ألغاز وذكاء وتسالي
- 📸 جلب صور من Pinterest
- 🧠 نصائح سنفورية يومية
- 🏆 نظام نقاط للمستخدمين
- 🇸🇩 مطور من السودان
